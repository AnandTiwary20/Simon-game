# Simon-game
This is the game based on Learning concept of my JS


A fun and interactive memory game built using HTML, CSS, and JavaScript. The classic Simon Game challenges users to remember and repeat an ever-increasing sequence of colors and sounds.


 How to Play-
Press any key to start the game.

Watch the sequence of colors that flash.

Repeat the sequence by clicking the buttons in the correct order.

The sequence gets longer each round. Try to keep up!

If you make a mistake, the game is over, and you can restart.


 Tech Stack-
HTML – Structure of the game

CSS – Styling and responsive layout

JavaScript – Game logic and interactivity

 Features-
Responsive design

Sound feedback for each button

Dynamic difficulty progression

Game restart on wrong move
